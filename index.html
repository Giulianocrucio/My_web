Technology Stack

Neural Networks

Built from scratch with 3 hidden layers of 16,32,16 neurons each, using ReLU activation

Genetic Algorithm

Roulette wheel selection with crossover breeding and adjustable mutation rates

Matter.js Physics

Realistic physics simulation for rocket movement and collision detection

JavaScript & HTML5

Pure JavaScript implementation with Canvas rendering for smooth visualization

Implementation Details

Rocket Design

  • Three thrusters: left, right, and center
  • Binary thruster states (on/off)
  • Physics-based movement and rotation
  • Collision detection with ground

Neural Network Architecture

  • Input: distance, orientation, angular velocity...
  • 3 hidden layers with 16,32,16 neurons
  • ReLU activation function
  • Output: 3 binary thruster controls

Evolution Process

  • Fitness scoring based on landing quality
  • Roulette wheel parent selection
  • Crossover breeding from two parents
  • Adjustable mutation factor

Visualization Features

  • Real-time performance graphs
  • Population statistics tracking
  • Interactive parameter controls
  • Multi-generation comparisons
